## Working with Block Themes

LexiaDesign mostly works like any other traditional WordPress theme. You can create posts and pages just like you always have. However, as a block theme, LexiaDesign also supports powerful new features like the site editor, patterns, global styles, and more. 

Here is excerpt regarding the definition of block theme from [WordPress](https://wordpress.org/documentation/article/block-themes/)

A block theme is a theme that uses blocks for all parts of a site, including navigation menus, header, content, and site footer. These themes are built for the newest features coming to WordPress that allow you to edit and customize all parts of your site. This allows you to switch up key parts of your theme without ever needing to change themes.

With a block theme, you can place and edit blocks outside your content using the Template Editor or Site Editor. You can use blocks in templates provided by a theme or plugin. For example, an archive page or a 404 page. You can also create custom templates.

### Site Editor

The current and simple way to build beautiful websites with WordPress is to use Site Editor. You can use a full suite of drag-and-drop design tools, blocks, and patterns.

Here is to building WordPress without an extra page builder.

To edit your site via the site editor, go to `Appearance → Editor`.

Here, you can

*   Create and Edit Templates
*   Customize Website Styles
*   Color Palette Customization
*   Typography Customization
*   Block Styles Customization
*   Create Menus
*   And Much More

#### Creating page designs with patterns

Insert LexiaDesign's pre-designed full-page patterns onto any page to achieve the layouts you see on the theme demo [LexiaDesign theme demo](https://lexiadesign.com).

Access the editor sidebar and apply the "No Page Title" template. This template removes the default title, allowing the full-page pattern to take center stage.

Since this template removes the title, remember to include an H1 heading within your design for optimal SEO practices.

### Global Styles

Imagine a place where you can easily control the look and feel of your entire website in one go. That's the power of Global Styles, a user-friendly interface within the WordPress Site Editor.

Fine-tune Every Detail:

*   Typography: Choose the perfect fonts and styles for your headings, body text, and more.
*   Colors: Set the color scheme for your buttons, links, and other website elements to match your brand identity.
*   Layouts: Define default settings for spacing, margins, and other layout aspects to create a cohesive visual style.

Powered by `theme.json`:

Behind the scenes, Global Styles relies on a file called theme.json located in your theme's root directory. This file acts as the blueprint, allowing you to define styles that can be applied globally across your site or even to specific blocks.

Benefits of Global Styles:

*   Effortless Customization: Make site-wide style changes without editing individual pages or blocks.
*   Cohesive Design: Ensure consistency in your website's appearance for a professional and polished look.
*   Simplified Workflow: Save time and streamline your design process by managing styles in one central location.

In essence, Global Styles empowers you to take control of your website's aesthetic with ease, allowing you to create a visually appealing and consistent online presence.

### Template Parts

Imagine building blocks for your website. That's what template parts are! These reusable sections, like headers, footers, and sidebars, appear across multiple pages, saving you time and effort.

The Benefits of Template Parts:

*   Effortless Updates: Make a change to a template part, and it instantly reflects on all pages that use it.
*   Global Consistency: Maintain a uniform look and feel throughout your website with ease.
*   Simplified Design: Focus on creating individual content while keeping common elements consistent.

Think of template parts as the foundation of your website's design, allowing you to build upon them with unique content for each page.

### Patterns

Building websites just got faster! Forget starting from scratch, WordPress creators can now use pre-designed page sections and full layouts called patterns. These building blocks are easily accessible through the block inserter, allowing you to quickly design stunning pages and even entire websites within the WordPress Site Editor.

You can access LexiaDesign's patterns via the block inserter on posts, pages, or in the site editor. 

### Export Your Site

Ready to share your masterpiece? Once you've built and customized your website using the Site Editor, you can easily export it for use on another site.

Here's how:

*   Head to the Site Editor.
*   Look for the Options menu in the top right corner.
*   Under the Tools heading, click on Export.

That's it! WordPress will take care of the rest, creating a compressed file (ZIP) containing all your customizations. You can then import this file onto another website to replicate your stunning creation.
